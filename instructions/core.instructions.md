# Core Rules (Always Loaded)

## Non-negotiable
1) Work strictly issue-by-issue using `.github/issues/*.md`.
2) Do not implement anything not requested by the current issue.
3) Bun-first: run scripts/tests with Bun.
4) Docker is OPTIONAL. Never require Docker to run locally.
5) Runtime logs: PT-BR. Comments/docblocks: EN.
6) Every issue must update `docs/evidence/*`.

## Evidence format
- Commands executed
- Outputs or placeholders for screenshots
- URLs when deployed
